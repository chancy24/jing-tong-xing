## 非功能检查

- `index.html`、`admin.html`、`data-policy.html`、`operation.html`、`about.html` 可直接打开；
- 所有 JS 和 CSS 路径正确；
- localStorage 空数据不报错；
- 页面刷新后固定通勤、收藏、历史、绿色积分仍保留；
- 后台不展示个人完整轨迹，只展示模拟和聚合统计。
